## Projects

### 🏥 [SmartClinic — Clinic Management System](https://github.com/muhammadammarkalim/SmartClinicSolution)

> A full-stack clinic management web application built with **Blazor Server** and **.NET 9**.

- **Description:** A multi-role clinic management platform featuring separate dashboards for Admins, Doctors, and Patients with full appointment, consultation, and revenue management.
- **Tech Stack:** Blazor Server, .NET 9, ASP.NET Core Identity, Entity Framework Core, SQL Server, Bootstrap 5.
- **Key Features:**
  - Role-based access control (Admin / Doctor / Patient) via ASP.NET Core Identity
  - Appointment booking and management system
  - Doctor profiles, consultations, and patient history tracking
  - Admin revenue analytics dashboard
  - Secure authentication with session management
- **Milestone:** Demonstrates full-stack .NET development with a code-first EF Core approach and real-world RBAC implementation.
- [🔗 View Repository](https://github.com/muhammadammarkalim/SmartClinicSolution)

### ✈️ [PK Mini Travel Planner](https://github.com/muhammadammarkalim/Mini-Travel-Planner)

- **Tech Stack:** C++, File I/O, Data Structures & Algorithms (DSA).
